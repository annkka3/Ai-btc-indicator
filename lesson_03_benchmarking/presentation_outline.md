### Слайд 3: Выбранные конкуренты
- **Конкурент 1**: TradingView (прямой/косвенный)
- **Конкурент 2**: Glassnode (косвенный)
- **Конкурент 3**: Сигнал-каналы в Telegram (прямой)
- **Обоснование выбора**: Различные типы конкурентов для комплексного анализа

---

### Слайд 4: TradingView - Обзор
- **Позиционирование**: "Лучшая платформа для технического анализа"
- **Цены**: $14.95-59.95/мес
- **Функционал**: Технические индикаторы, графики, Pine Script
- **ЦА**: Профессиональные трейдеры
- **Преимущества**: Профессиональные графики, социальная сеть
- **Недостатки**: Нет ML-прогнозов, сложность, высокая цена

---

### Слайд 5: Glassnode - Обзор
- **Позиционирование**: "Интеллектуальная аналитика для криптовалют"
- **Цены**: $29-799/мес
- **Функционал**: Ончейн-метрики, дашборды, API
- **ЦА**: Профессиональные аналитики
- **Преимущества**: Качественные ончейн-данные, профессиональная аналитика
- **Недостатки**: Нет ML-прогнозов, нет интеграции с Telegram, высокая цена

---

### Слайд 6: Сигнал-каналы - Обзор
- **Позиционирование**: "Торговые сигналы для криптовалют"
- **Цены**: $10-200/мес
- **Функционал**: Торговые сигналы, прогнозы направления
- **ЦА**: Розничные трейдеры
- **Преимущества**: Быстрота, простота, интеграция с Telegram
- **Недостатки**: Низкое качество сигналов, отсутствие объяснений, нет ML-прогнозов

### Слайд 8: Обязательные функции

- Вероятностный прогноз направления
- Объяснимые прогнозы с feature dump

---

### Слайд 9: Ключевые конкурентные преимущества
- **ML-прогнозы отклонения от тренда** - Уникальное преимущество
- **Объяснимые прогнозы** - Прозрачность использования признаков
- **Telegram-first** - Удобный доступ для розничных трейдеров
- **Скорость доставки** - ≤30-120с после закрытия бара
- **Режим рынка** - Контекст для интерпретации прогнозов
- **Доступная цена** - $9.99/мес (ниже конкурентов)
- **Инженерные практики** - Открыто задекларированные

---

### Слайд 10: Оптимальная модель ценообразования
- **Модель**: Freemium + Подписка
- **Бесплатный уровень**: 24ч-прогноз 1 раз/день
- **Платная подписка**: $9.99/месяц
- **Обоснование**: Доступная цена (ниже TradingView и Glassnode), конкурентное преимущество

---

### Слайд 11: Минимальные показатели (бенчмарки) - Продуктовые метрики
- **Activation**: ≥ 60% новых пользователей делают ≥1 запрос в первые 24ч
- **Engagement**: ≥ 3 карточки/пользователь/день
- **Retention**: D7 ≥ 40%, D30 ≥ 25%
- **Conversion to paid**: ≥ 10% активных пользователей
- **NPS/CSAT**: NPS ≥ 30, CSAT ≥ 4.0/5.0

---

### Слайд 12: Минимальные показатели (бенчмарки) - ML метрики
- **MAE-gain**: ≥ 5%/6%/8% (1ч/4ч/24ч) vs нулевой прогноз
- **AUC**: ≥ 0.58/0.60/0.62 (1ч/4ч/24ч)
- **Sharpe**: ≥ 1.0/1.1/1.2 (1ч/4ч/24ч)
- **Brier score**: ≤ 0.235/0.230/0.225 (1ч/4ч/24ч)
- **ECE**: ≤ 0.06 (калибровка вероятностей)

---

### Слайд 13: Минимальные показатели (бенчмарки) - Операционные метрики
- **Свежесть**: ≤ 30с после закрытия бара (1ч/4ч), ≤ 120с (24ч)
- **p95 инференса**: ≤ 2с
- **p95 доставки**: ≤ 5с после закрытия бара
- **Доступность**: ≥ 99% закрытых баров сопровождаются отчётом
- **Ошибки интеграций**: ≤ 1% 4xx/5xx/timeout в сутки

---

### Слайд 14: Release Gate
- **Условие релиза**: Выполнены **2 из 3** условий: {MAE-gain, AUC, Sharpe} на OOS
- **No-Go**: Если не выполнены 2/3 — перенос релиза и доработка
- **Цель**: Обеспечение качества прогнозов перед релизом

---
